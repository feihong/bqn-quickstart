# Cheatsheet

## Functions

### Conjugate

    +⟨1,2,3⟩ --> ⟨ 1 2 3 ⟩


### Add

    5+⟨1,2,3⟩ --> ⟨ 6 7 8 ⟩


### Negate

    -⟨1,2,3,4⟩ --> ⟨ ¯1 ¯2 ¯3 ¯4 ⟩


### Subtract

    "abcd"-32 --> "ABCD"


### Not

    ¬⟨0,1,0.25,0.75⟩ --> ⟨ 1 0 0.75 0.25 ⟩


### Span

    15¬10 --> 6
    9¬7 --> 3
    1¬3 --> ¯1


### Absolute Value

    |⟨¯3,0,4⟩ --> ⟨ 3 0 4 ⟩


### Modulus

    3|↕8 --> ⟨ 0 1 2 0 1 2 0 1 ⟩


### Enclose

    <"abc" --> ┌·       
               · "abc"  
                       ┘


### Less Than

    5<↕9 --> ⟨ 0 0 0 0 0 0 1 1 1 ⟩


### Merge

    "see Solo and Couple first" --> "see Solo and Couple first"


### Greater Than

    5>↕9 --> ⟨ 1 1 1 1 1 0 0 0 0 ⟩
    ⟨@,'a','Z'⟩>∞ --> ⟨ 1 1 1 ⟩


### Greater Than or Equal to

    5≥↕9 --> ⟨ 1 1 1 1 1 1 0 0 0 ⟩


### Rank

    =44 --> 0
    =↕4 --> 1
    =⟨⟩ --> 1
    =2‿2⥊↕4 --> 2
    =2‿3‿4⥊↕24 --> 3
    =⟨1‿2,3‿4‿5⟩ --> 1


### Equals

    'e'="Mesmerize" --> ⟨ 0 1 0 0 1 0 0 0 1 ⟩


### Reshape

    2‿3‿4⥊↕24 --> ┌─

## Modifiers

### Fold

    ⋈´"abcde" --> ┌─                          
                  · 'a' ⟨ 'b' ⟨ 'c' "de" ⟩ ⟩  
                                             ┘
    -´⟨1,2,3,4⟩ --> ¯2
    +´⟨⟩ --> 0
    ⌊´⟨⟩ --> ∞
    100-´⟨1,2,3,4⟩ --> 98


### Scan

    -`⟨1,2,3,4⟩ --> ⟨ 1 ¯1 ¯4 ¯8 ⟩
    100-`⟨1,2,3,4⟩ --> ⟨ 99 97 94 90 ⟩
